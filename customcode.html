Disable Custom Code

Adding third-party custom code can have unexpected results that affect your site's performance. If you've added custom code snippets, try disabling them to see if it fixes the problem.



To disable third-party code:


1. Go to Settings in your site's dashboard.

2. Click the Custom Code tab under Advanced Settings.

3. Disable the toggle(s) next to each custom code snippet you've added.

4. Reload your site or editor to see if the issue is resolved.
